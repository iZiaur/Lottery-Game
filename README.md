# Lottery Game

A simple lottery game made with React for learning JavaScript functions, random number generation, state handling, and digit sum logic.

You win the lottery when the sum of all digits of the ticket number is `15`.

Example:

357 → 3 + 5 + 7 = 15

## Features

- Random lottery ticket generation
- Winning condition based on digit sum
- React state management using `useState`


## Tech Stack

- React
- JavaScript
- CSS
